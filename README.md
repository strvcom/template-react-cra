## Prerequisites
- `node`
- `yarn`

## Usage
**Firstly, install all necessary dependencies:**
```sh
yarn install
```

**To start a development server:**
```sh
yarn start
```

**To build the app for production:**
```sh
yarn build
```

**To run tests:**
```sh
yarn test
```
